## Tecnologias utilizadas

- Framework Laravel
- Tailwind
- JavaScript
- SQLite

## Como rodar o projeto

- composer install
- npm install
- touch db.sqlite (Caso não tenha o banco ainda)
- php artisan migrate
- php artisan serve
- npm run dev
